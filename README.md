# app-noticias-ionic
Esta es una aplicación móvil para leer noticias, construida con las versiones más recientes de Ionic y Angular. 

El objetivo principal de este proyecto fue crear una app desde cero usando la arquitectura moderna de Angular (Standalone Components). La idea era dejar atrás los módulos tradicionales para tener un código mucho más limpio, ligero y fácil de mantener.

## ¿Qué hace la app?

* Se conecta a una API pública (News API) para traer titulares recientes en tiempo real.
* Permite guardar noticias en una pestaña de favoritos. Estos datos se quedan guardados en la memoria del teléfono.
* Tiene un diseño responsivo que incluye un modo oscuro personalizado.
* Funciona en la web, pero está lista para compilarse como una app nativa para Android o iOS.

## Tecnologías

* Ionic 8
* Nodejs 22
* Angular 20 (Standalone)
* TypeScript
* Capacitor
